# rusty-roguelike

A basic dungeon crawler in Rust. Journey through a forest or dungeon to claim th amulet of YALA (yet another lost amulet) for victory!

Based on the dungeon crawler from [Hands on Rust](https://pragprog.com/titles/hwrust/hands-on-rust/) with some additions.
